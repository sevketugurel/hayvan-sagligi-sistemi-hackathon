# Hayvan Sağlık Sistemi

Veteriner klinikleri için hayvan sağlık yönetim sistemi.

## Özellikler

- Hayvan kayıtları
- Sağlık takibi
- Randevu yönetimi
- Bildirim sistemi

## Kurulum

```bash
# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

## Teknolojiler

- Frontend: React/Vue/Angular
- Backend: Node.js
- Veritabanı: MongoDB/PostgreSQL
